## 快速开始指南

### 功能概览

这个企业门户网站模板管理系统包含以下核心功能：

#### 1. 模板库展示 (`/dashboard/analytics`)
- 浏览所有可用的企业门户网站模板
- 实时预览模板效果
- 生成新的模板实例

#### 2. 模板管理工作区 (`/dashboard/workspace`)
- 查看所有已生成的模板
- 编辑模板信息（名称、描述）
- 修改样式配置（颜色、字体）
- 管理导航菜单
- 删除不需要的模板

### 使用流程

#### 第一步：浏览模板
1. 访问 `/dashboard/analytics` 页面
2. 查看所有可用的模板卡片
3. 点击模板卡片进行选择

#### 第二步：预览模板
1. 选择一个模板后，点击"预览模板"按钮
2. 在弹出的预览窗口中查看模板效果
3. 包括导航栏、配色方案等信息

#### 第三步：生成模板
1. 选择一个模板后，点击"生成模板"按钮
2. 填写网站名称（必填）和网站描述（可选）
3. 点击"确认生成"按钮
4. 模板将被保存到本地存储

#### 第四步：管理模板
1. 访问 `/dashboard/workspace` 页面
2. 在左侧列表中选择要编辑的模板
3. 在右侧面板中查看模板详情
4. 点击"编辑模板"按钮进行修改

#### 第五步：编辑模板
在编辑模态框中，你可以：
- 修改网站名称和描述
- 调整颜色配置（主色、辅色、背景色、文字色）
- 添加、编辑或删除导航菜单项
- 点击"保存更改"保存所有修改

### 模板配置说明

#### 基本信息
- **网站名称**: 企业网站的名称
- **网站描述**: 网站的简短描述
- **基础模板**: 该模板基于哪个预设模板

#### 样式配置
- **主色**: 网站的主要品牌色
- **辅色**: 网站的辅助色
- **背景色**: 页面背景色
- **文字色**: 正文文字颜色

#### 导航菜单
- **菜单名称**: 导航项的显示名称
- **路由路径**: 导航项的链接路径
- **图标**: 菜单项的图标（可选）
- **子菜单**: 支持多级菜单结构

### 预设模板

系统内置了三个预设模板：

1. **现代科技风格** (template-1)
   - 适合科技公司
   - 蓝色系配色
   - 包含产品中心、解决方案等菜单

2. **商务优雅风格** (template-2)
   - 适合传统企业
   - 黑金配色
   - 包含公司简介、业务范围等菜单

3. **创意设计风格** (template-3)
   - 适合创意行业
   - 红青配色
   - 包含作品展示、服务项目等菜单

### 数据持久化

所有生成的模板数据都保存在浏览器的 `localStorage` 中：
- 存储键: `vben-generated-templates`
- 自动保存: 每次操作后自动保存
- 清除方式: 清除浏览器缓存会删除所有数据

### 常见操作

#### 添加新菜单项
1. 在编辑模板时，点击"+ 添加菜单项"
2. 输入菜单名称和路由路径
3. 点击"保存更改"

#### 编辑菜单项
1. 在导航菜单列表中，点击要编辑的菜单项旁的"编辑"按钮
2. 修改菜单名称、路由路径或图标
3. 点击"保存"

#### 删除菜单项
1. 在导航菜单列表中，点击要删除的菜单项旁的"删除"按钮
2. 菜单项将立即删除

#### 删除整个模板
1. 在 workspace 页面选择要删除的模板
2. 点击"删除"按钮
3. 确认删除操作

### 技术栈

- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **样式**: Scoped CSS
- **存储**: Browser localStorage

### 扩展建议

1. **添加更多模板**: 在 `src/views/templates/` 下创建新的模板文件夹
2. **后端集成**: 将 localStorage 替换为 API 调用
3. **导出功能**: 支持导出模板为 HTML/CSS 文件
4. **版本控制**: 记录模板的修改历史
5. **权限管理**: 添加用户权限和角色控制
6. **模板分享**: 支持模板分享和协作编辑

### 故障排除

#### 问题: 生成的模板没有保存
**解决**: 检查浏览器是否允许 localStorage，或清除浏览器缓存后重试

#### 问题: 预览模板时显示不正确
**解决**: 确保模板配置中的颜色值是有效的十六进制颜色代码

#### 问题: 菜单项编辑后没有保存
**解决**: 确保点击了"保存更改"按钮，而不仅仅是关闭编辑框
